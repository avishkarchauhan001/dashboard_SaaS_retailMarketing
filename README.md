# Xeno Dashboard

A premium, action-oriented dashboard design, a retail marketing SaaS platform. This project demonstrates a modern "Command Center" interface designed to help marketers understand account health and take immediate actions.

> **Note:** This is a **Frontend Design & Prototype**. It is built with React and uses static mock data to demonstrate the UI/UX concepts. There is no active backend integration.

## Features

*   **Account Health Pulse**: Real-time visualization of overall account status.
*   **Live Alerts Ticker**: Scrolling updates for urgent issues and opportunities.
*   **Active Campaigns Grid**: Management view for multi-channel campaigns.
*   **Top Segments**: Insights into high-value customer groups.
*   **Revenue Impact**: Visual tracking of revenue goals.
*   **Quick Actions**: One-click access to common tasks.
*   **Global Search**: Fast navigation to any asset.

## Tech Stack

*   **Framework**: React + Vite
*   **Styling**: Vanilla CSS (CSS Modules/Variables) for a custom premium look.
*   **Icons**: Emoji-based (for lightweight prototyping).

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:5173](http://localhost:5173) to view the dashboard.
